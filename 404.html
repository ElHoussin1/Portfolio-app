Dashboard Grafana
Description du projet :

Ce projet consiste à créer un tableau de bord interactif en utilisant Grafana pour la visualisation des données en temps réel et Prometheus pour la collecte des métriques. L'objectif est d'offrir des capacités d'observation avancées pour les systèmes de production.

Technologies utilisées :
  • Grafana
  • Prometheus
  • APIs REST
  • Outils de visualisation de données
Fonctionnalités clés :
  • Récupération des données en temps réel via Prometheus.
  • Tableau de bord personnalisable avec des graphiques et des widgets.
  • Interface utilisateur intuitive et accessible aux utilisateurs non techniques.
Medhead - PoC
Description du projet :

Medhead est un projet Proof of Concept (PoC) pour une application de gestion de données hospitalières, développée avec Spring Boot et Angular. Ce projet vise à simplifier la gestion des données des patients et des hôpitaux en temps réel, en intégrant des fonctionnalités comme la prise de rendez-vous et l'accès aux ressources médicales.

Technologies utilisées :
  • Spring Boot
  • Angular
  • APIs REST
  • Base de données relationnelle
Fonctionnalités clés :
  • Gestion des données hospitalières en temps réel.
  • Planification et suivi des rendez-vous patients.
  • Plateforme sécurisée et évolutive avec backend Spring Boot.
